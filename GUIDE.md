## 1. Créer un compte

Deux façons de créer un compte :

**Avec Google (recommandé — plus rapide)**
1. Cliquez sur **Continuer avec Google**
2. Choisissez votre compte Google dans la fenêtre qui s'ouvre
3. Votre espace est créé automatiquement

**Avec un email et un mot de passe**
1. Cliquez sur **Créer un compte** en bas de la page de connexion
2. Entrez votre email
3. Choisissez un mot de passe d'au moins 6 caractères
4. Confirmez le mot de passe
5. Cliquez sur **Créer mon compte**

> La première personne à créer un compte devient automatiquement l'**administrateur** de l'espace familial. Les autres membres rejoignent via un code d'invitation (voir section 9).

---

## 2. Se connecter

1. Accédez à l'application
2. Connectez-vous avec Google **ou** avec votre email + mot de passe
3. Vous arrivez directement sur la liste de vos boîtes

---

## 3. Créer une boîte

Une boîte correspond à une vraie boîte physique — carton, bac, valise, etc.

1. Sur la page d'accueil, cliquez sur **+ Nouvelle boîte**
2. Entrez le nom de la boîte (ex : « Décorations Noël », « Outils électriques », « Vêtements été »)
3. Cliquez sur **Créer**

La boîte reçoit automatiquement un numéro unique (#1, #2, #3…). Vous pouvez coller ce numéro sur la vraie boîte pour la retrouver facilement.

---

## 4. Ajouter un objet dans une boîte

1. Cliquez sur une boîte pour l'ouvrir
2. Cliquez sur **+ Ajouter**
3. Remplissez le formulaire :
   - **Nom** *(obligatoire)* — ex : « Cafetière rouge », « Perceuse Bosch »
   - **Description** *(optionnel)* — état, couleur, marque, détails utiles
   - **Photo** *(optionnel)* — prenez une photo directement avec la caméra ou choisissez dans votre galerie
4. Cliquez sur **Ajouter**

### Ajouter une photo

- **📷 Caméra** — ouvre directement l'appareil photo (idéal sur mobile)
- **🖼️ Galerie** — choisissez une image depuis votre appareil

Formats acceptés : JPEG, PNG, WebP, GIF, HEIC, HEIF — taille maximale : 10 Mo.

---

## 5. Modifier ou supprimer un objet

Dans une boîte, chaque objet affiché a deux boutons :

- **✏️ (crayon)** — ouvre le formulaire de modification, vous pouvez changer le nom, la description ou la photo
- **🗑️ (corbeille)** — demande une confirmation avant de supprimer définitivement l'objet

---

## 6. Modifier ou supprimer une boîte

Sur la page d'accueil, chaque boîte a deux boutons visibles au survol :

- **✏️** — modifier le nom de la boîte
- **🗑️** — supprimer la boîte et **tous ses objets** (une confirmation est demandée)

> Attention : supprimer une boîte supprime aussi tous les objets qu'elle contient. Cette action est irréversible.

---

## 7. Rechercher un objet

La barre de recherche est accessible en haut de toutes les pages.

1. Cliquez sur la barre de recherche
2. Tapez le nom ou une description de l'objet
3. Les résultats s'affichent en temps réel avec le nom de la boîte où se trouve chaque objet
4. Cliquez sur un résultat pour ouvrir directement la boîte correspondante

La recherche porte sur le **nom** et la **description** de tous les objets de votre espace.

---

## 8. Inviter un membre de la famille

> Seul l'**administrateur** de l'espace peut générer des codes d'invitation.

1. Cliquez sur l'icône **👥 membres** en haut à droite
2. Dans la section « Inviter un membre », cliquez sur **+ Générer un code d'invitation**
3. Un code de 8 caractères apparaît (ex : `AB3C7D2E`)
4. Cliquez sur **Copier** et partagez ce code avec le membre à inviter (SMS, message, etc.)

**Limites du code :**
- Valable **7 jours**
- Utilisable **une seule fois**
- L'espace est limité à **5 membres** au total

---

## 9. Rejoindre l'espace d'un proche

Si un membre de votre famille vous a partagé un code d'invitation :

1. Créez d'abord un compte (section 1)
2. Une fois connecté, accédez à la page de jointure — l'URL vous sera communiquée avec le code
3. Entrez le code de 8 caractères
4. Cliquez sur **Rejoindre**

Vous avez maintenant accès à l'espace familial et voyez toutes les boîtes.

> Si vous avez déjà créé votre propre espace vide avant de recevoir le code, vous pouvez quand même rejoindre l'espace d'un proche en utilisant le code directement.

---

## 10. Gérer les membres

La page membres (icône **👥**) affiche :

- La liste de tous les membres avec leur rôle (**Admin** ou **Membre**)
- Le badge **Admin** identifie le créateur de l'espace

**Retirer un membre (admin seulement)**
1. Cliquez sur **✕** à côté du membre à retirer
2. Confirmez le retrait

Le membre retiré perd immédiatement l'accès à l'espace et à toutes ses boîtes.

## 11. Installer l'app sur votre téléphone ou tablette

StorageApp est une PWA (Progressive Web App) — elle peut s'installer comme une vraie application, sans passer par l'App Store.

### Sur iPhone / iPad (Safari)
1. Ouvrez l'app dans **Safari**
2. Appuyez sur l'icône de partage **⎙** en bas de l'écran
3. Faites défiler et appuyez sur **« Sur l'écran d'accueil »**
4. Appuyez sur **Ajouter**

### Sur Android (Chrome)
1. Ouvrez l'app dans **Chrome**
2. Appuyez sur le menu **⋮** (trois points) en haut à droite
3. Appuyez sur **« Ajouter à l'écran d'accueil »** ou **« Installer l'application »**
4. Confirmez

L'app apparaît ensuite sur votre écran d'accueil comme n'importe quelle application.

## 12. Messages d'erreur courants

Si vous rencontrez une erreur, elle s'affiche avec un code entre parenthèses. Ce code permet à l'administrateur technique de retrouver rapidement la cause du problème.

| Code | Message affiché | Que faire |
|------|----------------|-----------|
| `ERR-AUTH-002` | Accès à votre espace refusé | Contactez l'administrateur de l'espace |
| `ERR-AUTH-005` | Email ou mot de passe incorrect | Vérifiez vos identifiants |
| `ERR-AUTH-006` | Email déjà utilisé | Connectez-vous ou utilisez un autre email |
| `ERR-AUTH-009` | Trop de tentatives | Attendez quelques minutes avant de réessayer |
| `ERR-BOX-002` | Impossible de créer la boîte | Vérifiez votre connexion internet |
| `ERR-ITEM-002` | Impossible de créer l'élément | La boîte a peut-être été supprimée, vous serez redirigé |
| `ERR-PHOTO-001` | Type de fichier non supporté | Utilisez JPEG, PNG ou WebP |
| `ERR-PHOTO-002` | Fichier trop volumineux | Réduisez la taille de l'image (max 10 Mo) |
| `ERR-PHOTO-003` | Échec de l'upload de la photo | Vérifiez votre connexion et réessayez |
| `ERR-INVITE-002` | Code d'invitation invalide | Vérifiez le code reçu (8 caractères) |
| `ERR-INVITE-003` | Code déjà utilisé | Demandez un nouveau code à l'administrateur |
| `ERR-INVITE-004` | Code expiré | Demandez un nouveau code (valable 7 jours) |
| `ERR-INVITE-005` | Espace complet | L'espace a atteint la limite de 5 membres |
| `ERR-FIREBASE-001` | Action non autorisée | Déconnectez-vous et reconnectez-vous |
| `ERR-FIREBASE-004` | Erreur réseau | Vérifiez votre connexion internet |

Si le problème persiste, notez le code d'erreur et signalez-le à votre administrateur.
